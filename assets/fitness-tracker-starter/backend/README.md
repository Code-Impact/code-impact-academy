# backend/

Your API lives here. It handles logic the front end shouldn't own
(e.g. streaks, weekly summaries, an optional AI insight).

- **Week 7:** build it and run it locally.
- **Week 8:** containerise with the Dockerfile in this folder.
- **Week 11:** ship the same image to AWS.

Pick your language/framework in `docs/ARCH.md` (e.g. FastAPI or Node/Express),
then have your agent scaffold it. Update the Dockerfile to match your choice.
